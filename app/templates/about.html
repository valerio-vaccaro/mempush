Project Description

MemPush is a lightweight Flask web application for managing and pushing Bitcoin transactions to the mempool. This tool provides a simple interface to submit raw transactions, track their status, and manually push them to the Bitcoin network.

‚ú® Key Features:
  • Submit raw Bitcoin transactions
  • Track transaction status (pending, confirmed, failed)
  • Manual transaction pushing to mempool
  • RESTful API for programmatic access
  • Transaction history management
  • Real-time status updates

API Documentation

üîå REST API Endpoints:
GET /api/transactions

Returns a JSON array of all known transactions ordered by creation date (newest first).

Example: curl -X GET http://localhost:5000/api/transactions
GET /api/transaction/<txid>

Returns complete transaction information for a specific transaction ID.

Example: curl -X GET http://localhost:5000/api/transaction/abc123...
POST /api/transaction

Submit a transaction ID to fetch and store the transaction from the mempool.

Request: {"txid": "your_txid_here"}
Example: curl -X POST http://localhost:5000/api/transaction -H "Content-Type: application/json" -d '{"txid": "abc123..."}'
POST /api/transaction/push

Submit a raw transaction in hex format and push it to the mempool.

Request: {"raw_tx": "your_hex_transaction_here"}
Example: curl -X POST http://localhost:5000/api/transaction/push -H "Content-Type: application/json" -d '{"raw_tx": "0100000001..."}'
üìä Response Status Codes:
  • 200 Success
  • 201 Created
  • 400 Bad Request (invalid input)
  • 403 Forbidden (unauthorized action)
  • 404 Not Found
  • 500 Server Error
